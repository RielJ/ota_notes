# Note Taking Application API

This is a simple note taking application API. It is built using Express + Typescript + Jest. It allows users to create, read, update and delete notes.

## Getting Started

### Prerequisites

- Node.js v20.0.0 or higher
- pnpm

### Installation

```bash
pnpm install
```

## Running the application

```bash
pnpm start
```

## Running the tests

```bash
pnpm test
```

## API Documentation

### Models

#### Note

- id (string) - a unique identifier auto-generated by the server
- title (string) - shouldn't exceed 100 characters
- body (string) - shouldn't exceed 1000 characters

```json
{
  "id": "string",
  "title": "string",
  "body": "string"
}
```

### Create a note

```http
POST /notes
```

#### Request Body

```json
{
  "title": "Note Title",
  "body": "Note Content"
}
```

#### Response

```json
{
  "id": "some-integer-id",
  "title": "Note Title",
  "body": "Note Content"
}
```

### Get all notes

```http
GET /notes
```

#### Response

```json
[
  {
    "id": "some-integer-id",
    "title": "Note Title",
    "body": "Note Content"
  }
  // ...
]
```

### Get a note

```http
GET /notes/:id
```

#### Response

```json
{
  "id": ":id",
  "title": "Note Title",
  "body": "Note Content"
}
```

### Update a note

```http
PUT /notes/:id
```

#### Request Body

```json
{
  "title": "New Note Title",
  "body": "New Note Content"
}
```

#### Response

```json
{
  "id": ":id",
  "title": "New Note Title",
  "body": "New Note Content"
}
```

### Delete a note

```http
DELETE /notes/:id
```

#### Response

```json
{
  "id": ":id",
  "title": "Note Title",
  "body": "Note Content"
}
```
